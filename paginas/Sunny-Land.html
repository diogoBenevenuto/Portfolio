Sunny Land Forest


Sunny Land Forest is a 2D platformer game in pixel art, made in Unity. Inspired by Super Mario, the game features challenging levels, enemies, and collectibles. Explore the forest, jump between platforms, and overcome obstacles in this fun adventure!

Project Information

  • Type of Project: Study Project
  • Tech: Unity
  • Platform: PC
  • Role: Programmer and Designer
