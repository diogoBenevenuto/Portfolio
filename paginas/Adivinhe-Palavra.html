Guess the Word


Guess the Word is a simple and fun game where you need to guess the secret word. When you start, you must guess the correct word, letter by letter, based on the given clue. With each correct letter, you progress and earn points, but be careful: if you make three mistakes, the game ends. With each new attempt, you can restart and challenge yourself again. Developed in React, this game is perfect for passing the time and testing your guessing skills!

Project Information

  • Type of Project: Study project
  • Tech: React
  • Platform: PC
  • Role: Programmer and Designer
