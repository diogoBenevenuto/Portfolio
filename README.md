# 💼 Portfólio - Diogo Benevenuto
Este é o meu portfólio pessoal, desenvolvido utilizando **JavaScript, HTML e CSS**, com o objetivo de apresentar meus projetos e habilidades como desenvolvedor.

## 🚀 Sobre o Projeto

O portfólio foi criado para reunir e organizar meus principais trabalhos de forma clara e acessível, destacando minhas áreas de atuação e experiência prática.

## 📂 Seções

O site está dividido nas seguintes áreas:

- 🎮 **Game Development**  
  Projetos de jogos desenvolvidos com foco em gameplay, mecânicas e prototipagem.

- ⚙️ **Backend Projects**  
  Projetos voltados para desenvolvimento backend, incluindo APIs, lógica de negócio e integração de sistemas.

- 🎨 **Designer**  
  Trabalhos visuais, incluindo modelagem 3D no Blender e criação de pixel art.

## 🛠️ Tecnologias Utilizadas

- JavaScript
- HTML5
- CSS3
